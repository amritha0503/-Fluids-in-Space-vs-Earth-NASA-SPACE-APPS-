# 📊 QUICK START: Adding Your NASA Dataset

## 🎯 In 3 Simple Steps

### Step 1: Copy Your Data File Here ⬇️

```
📁 fluids-in-space-react
  └── 📁 preprocessing
      └── 📁 raw_data          ← PUT YOUR CSV FILE HERE!
          └── your-nasa-data.csv
```

**Command:**
```powershell
Copy-Item "C:\path\to\your-nasa-data.csv" "preprocessing\raw_data\your-nasa-data.csv"
```

---

### Step 2: Run This Command ⬇️

```bash
cd preprocessing
python preprocess_nasa_data.py --input raw_data/your-nasa-data.csv --output ../src/data/processed.json --type marangoni
```

**Replace:**
- `your-nasa-data.csv` with your actual filename
- `marangoni` with your data type: `colloid`, `marangoni`, or `capillary`

---

### Step 3: See Your Data in the App! ⬇️

```bash
cd ..
npm start
```

Then open: **http://localhost:3000/learn-more**

Your data will appear in the charts! 📈

---

## 📋 What File Types?

✅ **CSV files** (`.csv`)
✅ **Excel files** (`.xlsx`) - convert to CSV first
✅ **Tab-delimited** (`.txt`, `.tsv`) - save as CSV


## 🆘 Problems?

**Error: "File not found"**
→ Check file is in `preprocessing/raw_data/`

**Error: "Column not found"**
→ Check your CSV has the right column names

**Need help with custom data?**
→ See `DATASET_INTEGRATION_GUIDE.md`
